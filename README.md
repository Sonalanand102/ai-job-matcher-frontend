# AI Semantic Resume Matcher - Frontend

Modern frontend for an AI-powered resume matching platform built with **Next.js**, **TypeScript**, and **Tailwind CSS**.

The application allows users to upload resumes, trigger AI-powered job matching, and visualize results generated through semantic search, hybrid retrieval, and LLM reranking.

---

## Live Demo

Frontend: https://ai-job-matcher-frontend-yuai.vercel.app

Backend: FastAPI (AWS EC2)

---

## Features

* Modern Responsive UI
* Resume Upload Interface
* AI Processing Pipeline Visualization
* Job Match Results Dashboard
* Match Score Visualization

* Mobile Friendly Design
